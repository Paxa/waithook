WaitHook

Web Hook to Websocket Transmiting Service

Waithook was build to ease receiving HTTP notifications in development and testing environment. It allows you to set web hook URL once and recieve requests from any location. The advantage of using WebSocket — it can work behind NAT or Firewall or Proxy.

1. Open WebSocket connection to ws://waithook.com/$your_unique_name
2. Send request to http://waithook.com/$your_unique_name
3. You got it!
